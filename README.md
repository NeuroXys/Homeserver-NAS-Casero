## ¿Qué es?
Un homeserver casero con piezas recicladas es, en esencia, un ordenador reaprovechado que se transforma en un servidor doméstico siempre encendido, pensado para centralizar servicios, datos y tareas dentro de casa sin depender de la nube ni de hardware nuevo.

### ¿En qué consiste?

Se parte de componentes de PC que ya no se usan, CPU y placa base antiguas, memoria DDR3, discos duros usados y se montan en una caja convencional.

Sobre ese hardware se instalará uno de los sistemas especializados siguientes:

* Proxmox VE: orientado a virtualización. Permite crear máquinas virtuales y contenedores (LXC/Docker) para separar servicios, probar sistemas operativos y gestionar todo desde una interfaz web.

* TrueNAS: centrado en almacenamiento. Convierte el equipo en un NAS completo con ZFS, control de integridad de datos, snapshots y compartición de archivos en red.

### ¿Para qué se utiliza?

 Este homeserver alojará varios servicios a la vez, por ejemplo:

* Servidor de archivos (copias de seguridad, nube privada tipo Nextcloud).

* Multimedia (Plex, Jellyfin).

* Servicios de red (Pi-hole, VPN, DNS local).

* Automatización y domótica (Home Assistant).

* Biblioteca digital en remoto.

## What is it?

A home server built with recycled parts is, essentially, a repurposed computer that is transformed into an always-on domestic server, designed to centralize services, data, and tasks within the home without relying on the cloud or new hardware.

### What does it consist of?

It is built using PC components that are no longer in use, older CPUs and motherboards, DDR3 memory, used hard drives, assembled into a standard case.

On this hardware, one of the following specialized systems will be installed:

* Proxmox VE: focused on virtualization. It allows the creation of virtual machines and containers (LXC/Docker) to separate services, test operating systems, and manage everything through a web interface.

* TrueNAS: focused on storage. It turns the system into a complete NAS with ZFS, data integrity control, snapshots, and network file sharing.

### What is it used for?

This home server will host multiple services at the same time, for example:

* File server (backups, private cloud such as Nextcloud).

* Multimedia (Plex, Jellyfin).

* Network services (Pi-hole, VPN, local DNS).

* Automation and home automation (Home Assistant).

* Remote digital library.
