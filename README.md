## ¿Qué es?
Un homeserver casero con piezas recicladas es, en esencia, un ordenador reaprovechado que se transforma en un servidor doméstico siempre encendido, pensado para centralizar servicios, datos y tareas dentro de casa sin depender de la nube ni de hardware nuevo.

### ¿En qué consiste?

Se parte de componentes de PC que ya no se usan, CPU y placa base antiguas, memoria DDR3, discos duros usados y se montan en una caja convencional.

Sobre ese hardware se instalará uno de los sistemas especializados siguientes:

* Proxmox VE: orientado a virtualización. Permite crear máquinas virtuales y contenedores (LXC/Docker) para separar servicios, probar sistemas operativos y gestionar todo desde una interfaz web.

* TrueNAS: centrado en almacenamiento. Convierte el equipo en un NAS completo con ZFS, control de integridad de datos, snapshots y compartición de archivos en red.

### ¿Para qué se utiliza?

 Este homeserver alojará varios servicios a la vez, por ejemplo:

* Servidor de archivos (copias de seguridad, nube privada tipo Nextcloud).

* Multimedia (Plex, Jellyfin).

* Servicios de red (Pi-hole, VPN, DNS local).

* Automatización y domótica (Home Assistant).

* Biblioteca digital en remoto.
